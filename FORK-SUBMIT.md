# Fork-and-submit — M10 Lab

1. From `LevelUp-Applied-AI/m10-l10` click **Use this template** (or
   **Fork**).
2. Clone your fork; create branch `lab-10-fastapi-kg-service`.
3. Implement the TODOs across `api/` and `web/`.
4. Push the branch; open a PR within your own fork.
5. Wait for the autograder; iterate until green.
6. Paste your PR URL into TalentLMS → Module 10 → Applied Lab.
